L'application a été crée avec Yeoman, elle est utilisable à partir de la comande grunt serve




Un site stylisé

Nous avons utilisé Bootstraps, les fonctionnalités principales de SASS, ainsi que plusieurs packages Bower afin de créer une application responsive

AngularJS

Notre application est fondée sur une architechture MVC, un filter est utilisé pour trier les données, la navigation est organisée par un Route Provider , et nous utilisons des directives

Fonctionnalités

Notre application est facile d'utilisation et propose différentes fonctionnalités telles que la recherche d'éléments, et l'affichage de graphiques
